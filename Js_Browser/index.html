** Javascript DOM manipulation **

The DOM is an in-memory representation of the active HTML document. Any changes made are local and do not affect the document stored on the webserver.

** Javascript Modules **

** Javascript Dom Selector **

1- This querySelector( ) method returns the first element in the document that matches the selector.
2- This querySelectorAll( ) method returns all elements in the document that match the selector.
3- This getElementById( ) method returns a specific element whose id matches the selector.
4- This getElementsByClassName( ) method returns all elements in the document that has the class specified in the selector.

** Event Handling **

1- Button click is an example of an event that your code can listen out for.
2- Icon tap is an example of an event that your code can listen out for.

** Object Notation - JSON **

When working with JSON it is common to convert it back to a JavaScript object to work with its properties. To do this you need to use the global built-in JSON object and its parse method.
